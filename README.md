# ⚛️ React Components & React Hooks

This repository contains my practice and learning exercises for **React Components** and **React Hooks**. It includes examples of creating reusable components and managing state using React Hooks.

### ✅ React Components

- JSX
- Props
- Component Reusability
- Component Composition

### ✅ React Hooks
- useState()
- useEffect()
- useRef()
- useContext() (Basics)
- Custom Hooks (Introduction)
